### \* DFS,BFS

> DFS

- 깊이 우선 탐색 알고리즘
- 최대한 멀리 있는 노드를 우선으로 탐색하는 방식으로 동작, 스택 자료구조 이용<br>

- 너비 우선 탐색 알고리즘
